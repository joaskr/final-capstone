## Usage
Steps to open the project locally: \
Step 1: fork the repo \
Step 2: `cd <your location>` \
Step 3: `git clone <link to your fork>` \
Step 4: `cd <folder with repo>` \
Step 5: `npm install` \
Step 6: `cd client` \
Step 7: `npm install` \
Step 8: `npm run build` \
Step 9: `cd ..` \
Step 10: `npm start`
